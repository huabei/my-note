创建标题

添加#号，#号的个数就是标题的层级

段落

^11e80d

要创建段落，使用空白行将一行或者多行文本进行分隔。

换行

在一行的末尾添加两个或多个空格，然后加一个换行符。

强调语法

  • 粗体

    两个星号或下划线将需要加粗的部分包括进去

  • 斜体

    用一个星号或下划线将需要斜体的部分包括进去

  • 同时粗体和斜体

    使用三个星号或下划线将需要加粗倾斜部分包括进去

引用语法

  • 块引用

    在段落前添加一个>符号创建块引用

  • 多个段落的块引用

    在[[Markdown#^11e80d|段落中间的空白行]]添加>符号

    A

  • 嵌套块引用

    在要嵌套的段落前

    加 >> 符号

列表

  • 有序列表

    1. 在每个列表项前添加一个数字并紧跟一个英文句点1.

    2. 数字不必按数学顺序排列

  • 无序列表

    • 在每个列表项前添加破折号-、星号*或加号+

  • 列表嵌套

    • 将嵌套元素缩进四个空格或一个制表符

代码语法

  • 代码

    • 将代码包裹在反引号中

    • 如果代码中包括反引号，用两个反引号包括起来

  • 代码块

缩进或用三个反引号包括

import os
print(os.get_pwd())

import os
print(os.get_pwd())

分割线

在单独的一行上使用三个或多个星号、破折号、或下划线

链接语法

链接文本放在中括号内，链接地址放在后面的括号内，链接title可选
