HTML 基础

文档类型定义

文档类型定义（Document Type Definition， DTD）指明了文档中所包含的HTML版本。网页文档里的第一行就是DTD语句，通常称作文档类型语句。HTML5的DTD如下：
<!DOCTYPE html>

网页模板

每一张页面都会包含DTD、html、head、title、meta和body元素。一个基本的HTML5网页模板如下所示：

<!DOCTYPE html>
<html lang="en">
<head>
<title>Page Title Gose Here</title>
<meta charset="utf-8">
</head>
<body>
... body text and more HTML tags go here ...
</body>
</html>

HTML 元素

html元素的目的是指明该文档是HTML格式。
开始的<html>标签放置再DTD的下一行，结束的</html>指明网页的结束位置，它放置在文档中所有的HTML元素之后。其属性可以用来指定文档中所含文本中所使用的语言。

四大元素head、title、meta和body

网页有两部分：标题和主体。标题部分包含描述网页文档的信息。主体部分包含实际标签、文本、图像以及其他被浏览器显示为网页的对象。

头部（Head）

头部的元素包含网页的title、描述文档的元信息meta标签以及对脚本和样式的引用等。

标题（title）

头部的第一个元素是title，它配置了在浏览器窗口标题栏里显示的文本。在页面上添加书签或打印网页时即访问这一元素。

元信息（meta）

meta元素描述网页的特性，例如字符集。
字符集是网页文档或其他文件中字母、数字和符号的内部表述，它们储存在计算机上，可以在网上传输。meta元素并不成对存在，如<meta charset="utf-8">。

主体（Body）

主体包含的文本和元素直接显示在浏览器所展现的网页上，所以也称作浏览器视窗（viewport）。使用主体部分的目的在于配置网页的内容。
