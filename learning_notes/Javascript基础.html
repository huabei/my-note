Javascript基础

Data Types

三种定义数据的方式：\[keyword\]\[name\]

  • 变量（Variables）
    • var：全局变量，可以再声明前调用
    • let：局部变量
  • 常量
    • const，不能够被改变

六种原始数据类型：string, number, bigint, boolean, undefined, symbol.

myNumber = 123;
myString = "123";
myBool = false;

算术操作符

+、-、\*、/、%（取余）

字符串

用单引号或双引号包括

格式化字符串

let myString1 = "Hello";
let myString2 = "World";

myString1 + myString2 + "!"; //HelloWorld!
myString1 + " " + myString2 + "!"; //Hello World!
myString1 + ", " + myString2 + "!"; //Hello, World!

`${myString1} ${myString2}!` //Hello World!
`${myString1}, ${myString2}!` //Hello, World!

Methods and Functions

一个函数其实是一个代码块，通过一个名字调用，这对于多次调用一个任务是方便的，而且是易读的。

function nameOfFunction() { // function definition
 // function definition/body
}

一个显示Hello， world！的函数：

function displayGreeting() {
  console.log('Hello, world!');
}

当我们需要调用一个函数的时候，只需要在函数名后加()即可，需要注意的是，在JS中函数的定义可以在调用之后定义，编译器会自动去寻找。

// calling our function
displayGreeting();

还有一种特殊的函数叫做方法，其是绑定在对象上的，如console.log('Hello, world!'),在这里console就是对象。

函数的最佳实践

  • 函数名要有意义，让人能够知道其作用。
  • 使用驼峰命名法
  • 一个函数解决一个特定的任务，不要繁琐。

给函数传递信息

通过参数的方式给函数传递信息

function name(param, param2, param3) {

}

可以将displayGreeting()函数更新为：

function displayGreeting(name) {
  const message = `Hello, ${name}!`;
  console.log(message);
}

调用函数：

displayGreeting('Christopher');
// displays "Hello, Christopher!" when run

给参数加默认值：

function displayGreeting(name, salutation='Hello') {
  console.log(`${salutation}, ${name}`);
}

调用函数：

displayGreeting('Christopher');
// displays "Hello, Christopher"

displayGreeting('Christopher', 'Hi');
// displays "Hi, Christopher"

Return values

让函数返回值：

return myVariable;

一个生成欢迎词的函数：

function createGreetingMessage(name) {
  const message = `Hello, ${name}`;
  return message;
}

调用函数：

const greetingMessage = createGreetingMessage('Christopher');

将函数作为参数：

function displayDone() {
  console.log('3 seconds has elapsed');
}
// timer value is in milliseconds
setTimeout(displayDone, 3000);

匿名函数

如果一个函数我们在代码中只需要调用一次，则使用匿名函数：

setTimeout(function() {
  console.log('3 seconds has elapsed');
}, 3000);

胖箭头函数（Fat arrow functions)

通过Fat arrow=>可以跳过function关键字：

setTimeout(() => {
  console.log('3 seconds has elapsed');
}, 3000);

三种传入函数作为参数的方法

如果函数要使用多次，则按照通常的方法构建。如果函数只调用一次，则使用匿名函数。至于使用胖箭头=>替代function关键字，均可。

Making Decisions

布尔值（Booleans）

布尔值只有两个：true或者false，布尔值可以帮助决定运行哪一行代码。

let myTrueBool = true;
let myFalseBool = false;

比较运算符

小于：< 小于等于：<= 大于：> 大于等于：>= 等于（数据类型可以不同）：5=='5'//true 不等于：5!='5'//true 强等于（数值和数据类型均相同）：5==='5'//false 不强等于：5!=='5'//true

If 语句

如果条件语句为真，则运行代码块中的内容：

if (condition){
    //Condition was true. Code in this block will run.
}

通常条件语句由逻辑运算符构成：

let currentMoney;
let laptopPrice;

if (currentMoney >= laptopPrice){
    //Condition was true. Code in this block will run.
    console.log("Getting a new laptop!");
}

If..Else 语句

如果条件语句为假，则运行代码块中的内容：

let currentMoney;
let laptopPrice;

if (currentMoney >= laptopPrice){
    //Condition was true. Code in this block will run.
    console.log("Getting a new laptop!");
}
else{
    //Condition was false. Code in this block will run.
    console.log("Can't afford a new laptop, yet!");
}

其可以选择性的跟在if语句后面

逻辑运算符

  • 逻辑与：&&
  • 逻辑或：||
  • 逻辑非：! 逻辑运算符用于处理两个布尔值

使用逻辑运算符

在if..else语句中使用逻辑运算符

let currentMoney;
let laptopPrice;
let laptopDiscountPrice = laptopPrice - (laptopPrice * .20) //Laptop price at 20 percent off

if (currentMoney >= laptopPrice || currentMoney >= laptopDiscountPrice){
    //Condition was true. Code in this block will run.
    console.log("Getting a new laptop!");
}
else {
    //Condition was true. Code in this block will run.
    console.log("Can't afford a new laptop, yet!");
}

取反

通过!可以对条件取反：

if (!condition) {
  // runs if condition is false
} else {
  // runs if condition is true
}

三元表达式

三元表达式可以代替简单的if..else语句：

let variable = condition ? <return this if true> : <return this if false>

具体使用：

let firstNumber = 20;
let secondNumber = 10
let biggestNumber = firstNumber > secondNumber ? firstNumber: secondNumber;

Arrays and Loops

数组是一种结构化的数据形式：

let myArray = [];

数组中的值以逗号隔开，可以包含不同类型的值：

let iceCreamFlavors = ["Chocolate", "Strawberry", "Vanilla", "Pistachio", "Rocky Road"];

数组中的值分配了一个索引，这个索引是值到开始位置的距离，可以通过索引读取，改变或插入新值。

let iceCreamFlavors = ["Chocolate", "Strawberry", "Vanilla", "Pistachio", "Rocky Road"];
iceCreamFlavors[2]; //"Vanilla"
iceCreamFlavors[4] = "Butter Pecan"; //Changed "Rocky Road" to "Butter Pecan"
iceCreamFlavors[5] = "Cookie Dough"; //Added "Cookie Dough"

也可以通过数组操作增加新值：

iceCreamFlavors.push('Cookie Dough')

通过length属性获取数组长度：

let iceCreamFlavors = ["Chocolate", "Strawberry", "Vanilla", "Pistachio", "Rocky Road"];
iceCreamFlavors.length; //5

Loops

循环对于处理重复工作，减少代码量有重要作用。在JS中，循环的本质是：loop over data

For 循环

for循环需要三个部分： - 计数器：一个保存循环次数的变量 - 条件语句：用于判断终止的条件 - 迭代语句：在每个循环后执行的语句，用来改变计数器

// Counting up to 10
for (let i = 0; i < 10; i++) {
  console.log(i);
}

While 循环

//Counting up to 10
let i = 0;
while (i < 10) {
 console.log(i);
 i++;
}

循环和数组

数组经常在循环中使用，因为大多数时候循环需要判断数组的长度来终止，而且计数器可以索引数组中的元素。

let iceCreamFlavors = ["Chocolate", "Strawberry", "Vanilla", "Pistachio", "Rocky Road"];

for (let i = 0; i < iceCreamFlavors.length; i++) {
  console.log(iceCreamFlavors[i]);
} //Ends when all flavors are printed
